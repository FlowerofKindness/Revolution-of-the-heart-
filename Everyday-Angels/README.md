# Everyday Angels

Intuitive messages, affirmations, and symbolic storytelling for grounding support.

## Contents

This section houses:
- **Affirmations** — Daily or thematic affirmations
- **Intuitive Messages** — Wisdom & guidance for difficult moments
- **Symbolic Storytelling** — Short stories that anchor meaning

## Getting Started

Structure your content:

```
Everyday-Angels/
  ├── affirmations/
  │   ├── daily-affirmations.md
  │   ├── healing-affirmations.md
  │   └── presence-affirmations.md
  ├── intuitive-messages/
  └── stories/
```

Each file can be a collection or individual pieces.
