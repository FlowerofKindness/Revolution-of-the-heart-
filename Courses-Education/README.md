# Courses & Education

Trauma-aware educational offerings for practical rebuilding and autonomy.

## Contents

This section houses:
- **Courses** — Full curriculum with modules & lessons
- **Resources** — Templates, worksheets, tools
- **Guides** — How-to and reference materials
- **Practicum** — Applied learning & exercises

## Getting Started

Structure your courses:

```
Courses-Education/
  ├── course-001/
  │   ├── README.md
  │   ├── module-1/
  │   │   ├── lesson-1.md
  │   │   └── lesson-2.md
  │   └── module-2/
  ├── resources/
  │   ├── templates/
  │   └── worksheets/
  └── guides/
```

Each course should have:
- Course overview
- Learning objectives
- Modules (1-3 lessons each)
- Practicum exercises
