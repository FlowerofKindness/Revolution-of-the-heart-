# reBElious ME

Transforming survival into statement through creative expression.

## Contents

This section houses:
- **Writing** — Poems, essays, stories from the heart
- **Visuals** — Art, graphics, design
- **Wearable Art** — Symbolic clothing, accessories, statements

## Getting Started

Organize your work like this:

```
reBElious-ME/
  ├── writing/
  │   ├── poems/
  │   ├── essays/
  │   └── stories/
  ├── visuals/
  │   ├── art/
  │   └── graphics/
  └── wearable-art/
```

Each folder can contain `.md` documentation and asset files (images, etc.).
