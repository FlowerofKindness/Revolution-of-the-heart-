# 444-Generations Jewelry

Legacy work honoring generational healing through vintage jewelry resale.

## Contents

This section houses:
- **Inventory** — Curated vintage pieces with stories
- **Sourcing** — How pieces are selected & ethically sourced
- **Stories** — The generational & personal meaning of each piece
- **Shop Information** — Listing details, pricing, how to purchase

## Getting Started

Structure your content:

```
444-Generations-Jewelry/
  ├── inventory/
  │   ├── piece-001.md
  │   └── piece-002.md
  ├── stories/
  ├── sourcing.md
  └── shop.md
```

Each piece can have:
- Description
- Photo(s)
- Story/meaning
- Pricing
- Status (available/sold)
