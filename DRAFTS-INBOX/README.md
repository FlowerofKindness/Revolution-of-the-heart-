# DRAFTS & INBOX

This is your staging area. Dump all raw content here, then we'll sort & organize it into the right sections.

## How to add content:

1. **Copy-paste from ChatGPT/Google Docs/Notion** into new `.md` files here
2. **Upload markdown files** from your computer to this folder
3. **Label files clearly** so we know what they are:
   - `philosophy-draft.md`
   - `affirmations-batch-1.md`
   - `healing-story.md`
   - etc.

## Organization happens next

Once everything is here, we'll:
- Sort by section
- Edit & refine
- Move to the right places
- Delete this folder
